# vue-playground

기존 프로젝트의 낮은 버전 vue.js를 최신 버전의 vue.js로 변경하고
기존 MPA방식을 SPA방식으로 변경하여 성능을 개선 시킨다.

## Skills
- vue 3
- typescript
- vite 
- vue 3 setup/composition-api 
- bootstrap-vue-3
- pinia (store)
- axios
- lodash
- v-calendar
- crypto-js


### 설치

```sh
npm install
```

### 실행

```sh
npm run dev
npm run prod
```
