# vue-playground

This template should help get you started developing with Vue 3 in Vite.

## Skills
- vue 3
- typescript
- vite 
- vue 3 setup/composition-api 
- bootstrap-vue-3
- pinia (store)
- axios
- lodash
- v-calendar
- crypto-js


### 설치

```sh
npm install
```

### 실행

```sh
npm run dev
npm run prod
```
